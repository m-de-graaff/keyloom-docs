# CLI

Use the CLI to initialize, migrate, scan routes, and scaffold UI.

```bash
keyloom init [...flags]
keyloom migrate [...flags]
keyloom routes [...flags]
keyloom doctor [...flags]
keyloom generate migration
keyloom ui setup
keyloom ui add <sign-in|user-button|org>
```
