Authenticate with X (Twitter) using OAuth 2.0.

<Steps>

### 1) X developer app

- Redirect URI: `https://YOUR_DOMAIN/api/auth/callback/x`
- Copy Client ID and (optional) Client Secret

```bash
# .env.local
X_CLIENT_ID=...
X_CLIENT_SECRET=... # optional for some flows
```

### 2) Configure provider

```ts
// keyloom.config.ts
import { PrismaAdapter } from '@keyloom/adapters'
import { PrismaClient } from '@prisma/client'
import { X } from '@keyloom/providers'

const db = new PrismaClient()
export default {
  adapter: PrismaAdapter(db),
  providers: [
    X({
      clientId: process.env.X_CLIENT_ID!,
      clientSecret: process.env.X_CLIENT_SECRET, // optional
    }),
  ],
}
```

### 3) UI

</Steps>

<Callout>
Ensure your X app has the correct OAuth 2.0 scopes (e.g., `tweet.read users.read offline.access`) configured to match your usage.
</Callout>
