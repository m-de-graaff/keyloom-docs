# Providers

Keyloom supports first-class OAuth providers via @keyloom/providers.

**Available Providers:**

- **GitHub** - Developer platform authentication
- **Google** - Google accounts and Workspace integration
- **Discord** - Gaming and community platform authentication
- **Apple** - Sign in with Apple for iOS and web
- **Auth0** - Enterprise identity platform with SSO
- **GitLab** - GitLab.com and self-hosted instances
- **Microsoft** - Azure AD, Office 365, and Microsoft accounts
- **X** - X (formerly Twitter) social authentication

## Quick start

```ts title="keyloom.config.ts"
import { defineKeyloom } from "@keyloom/core";
import github from "@keyloom/providers/github";
import google from "@keyloom/providers/google";

export default defineKeyloom({
  // ...adapter, baseUrl, session
  providers: [
    github({
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
    }),
    google({
      clientId: process.env.GOOGLE_CLIENT_ID!,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
    }),
  ],
});
```

Callback URL format: `${baseUrl}/api/auth/callback/<providerId>`
