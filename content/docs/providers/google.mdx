# Google Provider

<Steps>
1. Create OAuth credentials
- Google Cloud Console - APIs & Services - Credentials
- Application type: Web application
- Authorized redirect URI: `${BASE_URL}/api/auth/callback/google`

2. Set environment variables
```env
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
```

3. Add to configuration
```ts title="keyloom.config.ts"
import google from "@keyloom/providers/google";

export default defineKeyloom({
  // ... adapter, baseUrl, session
  providers: [
    google({ clientId: process.env.GOOGLE_CLIENT_ID!, clientSecret: process.env.GOOGLE_CLIENT_SECRET! }),
  ],
});
```
</Steps>

<Callout>
If you enable additional Google scopes, handle them by extending your user model and adapter mapping.
</Callout>
