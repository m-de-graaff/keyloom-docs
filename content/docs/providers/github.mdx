# GitHub Provider

<Steps>
1. Create an OAuth app
- Go to GitHub Developer Settings â†’ OAuth Apps
- Homepage URL: your app URL
- Authorization callback URL: `${BASE_URL}/api/auth/callback/github`

2. Set environment variables
```bash
GITHUB_CLIENT_ID=...
GITHUB_CLIENT_SECRET=...
```

3. Add to configuration
```ts title="keyloom.config.ts"
import github from "@keyloom/providers/github";

export default defineKeyloom({
  // ... adapter, baseUrl, session
  providers: [
    github({ clientId: process.env.GITHUB_CLIENT_ID!, clientSecret: process.env.GITHUB_CLIENT_SECRET! }),
  ],
});
```
</Steps>

<Callout>
When using the App Router handler at app/api/auth/[...keyloom]/route.ts, the callback path is automatically routed.
</Callout>
