Use this section to diagnose common issues with Keyloom setup and runtime.

## Topics

- CSRF errors on POST requests
- OAuth callback and provider errors
- Cookie domains, SameSite, and Secure flags
- JWT strategy issues (issuer, keys, TTL)
- Database adapter and Prisma problems

<Callout>
Start with `keyloom doctor` to validate environment, routes, and secrets. Many issues are surfaced there.
</Callout>
