# Environment Variables

## Core/Next.js
- NEXT_PUBLIC_APP_URL: Public base URL (e.g., https://app.example.com)
- KEYLOOM_JWT_JWKS_URL: JWKS endpoint URL for verifying access tokens (JWT strategy)
- KEYLOOM_JWT_ISSUER: Expected token issuer (optional but recommended)
- KEYLOOM_JWT_AUDIENCE: Expected audience (optional)
- KEYLOOM_JWT_CLOCK_SKEW_SEC: Allowed clock skew in seconds (default 60)
