# CLI

The Keyloom CLI helps you get started quickly with sensible defaults and optional templates.

## Install and run

Run with your package manager:

```bash
npx keyloom init
# or
pnpm dlx keyloom init
```

## What it does

- Detects your package manager and framework
- Creates `keyloom.config.ts`
- Optionally adds Next.js middleware and an auth route handler
- Optionally installs provider and adapter packages

## Flags

- `--no-install` - skip installing dependencies
- `--force` - overwrite existing files
- `--providers=github,google` - preconfigure providers
- `--adapter=prisma|drizzle|memory` - choose adapter

## Example session

