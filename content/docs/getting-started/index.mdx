- Installation: /docs/getting-started/installation

- Comparison: /docs/getting-started/comparison
- Tutorials: /docs/tutorials

<Callout>
  Keyloom is TypeScript-first with production defaults. You need:
  <ul>
    <li>An adapter (PrismaAdapter for production)</li>
    <li>NEXT_PUBLIC_APP_URL and AUTH_SECRET (base64url, 32 bytes decoded)</li>
    <li>
      A single Next.js API handler via <code>createNextHandler</code>
    </li>
  </ul>
</Callout>
