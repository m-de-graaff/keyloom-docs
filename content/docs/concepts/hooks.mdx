# Server Helpers

Keyloom does not ship React client hooks. Use server helpers from @keyloom/nextjs for predictable, secure access to auth in the App Router.

## getSession and getUser
```ts
import { getSession, getUser } from "@keyloom/nextjs";
import config from "@/keyloom.config";

export default async function Page() {
  const { session, user } = await getSession(config);
  // ... render
}
```

## guard
```ts
import { guard } from "@keyloom/nextjs";
import config from "@/keyloom.config";

export default async function AdminPage() {
  await guard({ visibility: "role:admin", org: "required", redirectTo: "/sign-in" }, config);
  return <div>Admin content</div>;
}
```

<Callout>
For JWT sessions, the helpers will auto-configure using your baseUrl to fetch JWKS and verify tokens.
</Callout>
